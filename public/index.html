DiceBear Avatar Service

A service for generating and managing custom avatars

API Endpoints

GET /api/avatar/:userId

Retrieve an avatar for a user

Query parameters:

  • size: Image size in pixels (default: 128)
  • format: Image format - svg, png, webp (default: svg)
  • style: Optional style override
POST /api/auth/token

Generate an edit token (protected with API key)

POST /api/editor/:userId

Save avatar preferences (requires valid token)

GET /api/health

Check the health of the service
